# Guanajuato RP App

> Vue.js x Vuetify

## Commande pour faire tourner le projet :

- `npm install`
  - Download et installer les packages
- `npm run serve`
  - Lancer le serveur de dévelopement avec hot reload
- `npm run build`
  - Build et minify l'app pour production
